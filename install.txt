How to install the Pterodactyl Argon theme:
1- Upload the public and resources folder to your Pterodactyl installation directory. (/var/www/pterodactyl by default)
2- Modify your .env file to set the THEME property to "argon".
3- Run the following commands in your Pterodactyl installation directory. (/var/www/pterodactyl by default):
